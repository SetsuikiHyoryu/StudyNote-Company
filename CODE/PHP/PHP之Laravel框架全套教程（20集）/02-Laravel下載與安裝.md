## Laravel環境的搭建

- 要求的PHP版本：大於5.5.9

---

## 使用IDE打開Laravel

- 代碼寫在`/app`文件夾中
- `/app/Http/Controllers`文件夾就是MVC的C
- 文件的啟動和自動加載在`/bootstrap/cache`
- 配置文件的目録：`/config`
- 數據庫：`/database`
- `public`：包含入口和項目静態支援文件(img, css等)
- `resources`：包含視圖和原始支援文件
